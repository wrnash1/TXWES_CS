### 4. Quiz 8 (Question Bank)
**Question 1:** A user cannot reach a website by its name, but can reach it by typing its IP address. Which tool should the administrator use to troubleshoot this?
A) ping   B) tracert   C) nslookup   D) arp
*   **Correct:** C) nslookup
*   **Distractor Analysis:** The issue is clearly DNS (name resolution). `nslookup` queries DNS servers directly. `ping` and `tracert` test Layer 3 connectivity, which is already confirmed working via the IP.
