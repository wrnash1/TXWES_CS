### 2. Video Script 3.2: Virtualization vs. Cloud Computing
**Estimated Duration:** 5 minutes
**Visual:** A diagram showing a hypervisor managing VMs on the left, and a cloud data center on the right.
**[Alt-text: Left side: A single physical server running a Hypervisor and three VMs, labeled 'Virtualization'. Right side: A globe connected to multiple massive data centers, labeled 'Cloud Computing'.]**
**Audio:** "Many people confuse virtualization and cloud computing. Virtualization is the technology—it's what you're doing right now in your lab using VirtualBox. You are abstracting hardware. Cloud Computing takes virtualization and adds a self-service management layer over the internet. If you have to ask IT to build a VM for you, that's virtualization. If you log into a web portal, swipe a credit card, and build a VM yourself in 10 seconds, that's Cloud Computing."
