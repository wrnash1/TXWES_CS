# Online Course Map: CIS 3321 - Network Administration
**Target Certification:** CompTIA Network+ (N10-008)

## Overview
This online course map outlines the modular transition of CIS 3321 to a fully asynchronous format. The structure is designed to promote micro-learning, hands-on application via local virtual labs, and certification readiness through distractor-analyzed question banks.

---

## Module Architecture

### Module 1: Networking Fundamentals (Domain 1.0)
- **Learning Objectives:** Compare OSI and TCP/IP models, identify network topologies.
- **Micro-Learning Video Assets (5-7 mins):**
  - *Video 1.1: The OSI Model Demystified* (Includes Alt-text for OSI layer diagrams)
  - *Video 1.2: Physical vs. Logical Topologies*
- **Reading Guide:** Chapter 1 Summary & Key Terms.
- **Lab Assignment:** Local VirtualBox / Packet Tracer - Build a basic LAN.
- **Assessment:** Quiz 1 (Domain 1.0) with Distractor Analysis.

### Module 2: Network Implementations (Domain 2.0)
- **Learning Objectives:** Configure switches and routers, explain IPv4 vs IPv6.
- **Micro-Learning Video Assets:**
  - *Video 2.1: Switching Concepts and VLANs*
  - *Video 2.2: IPv4 Subnetting Basics* (Includes Alt-text for binary math tables)
- **Reading Guide:** Chapter 2-3 Interactive Review.
- **Lab Assignment:** Configuring VLANs in a virtual switch.
- **Assessment:** Quiz 2 (Domain 2.0) with Distractor Analysis.

### Module 3: Wireless and Cloud (Domains 2.0 & 3.0)
- **Learning Objectives:** Wi-Fi standards, cloud service models (IaaS, PaaS, SaaS).
- **Micro-Learning Video Assets:**
  - *Video 3.1: 802.11 Standards Breakdown*
  - *Video 3.2: Virtualization vs. Cloud Computing*
- **Reading Guide:** Chapter 5 & 9 Highlights.
- **Lab Assignment:** Setting up a hypervisor (VirtualBox locally).
- **Assessment:** Quiz 3 with Distractor Analysis.

### Module 4: Network Security (Domain 4.0)
- **Learning Objectives:** CIA Triad, common attacks, mitigation techniques.
- **Micro-Learning Video Assets:**
  - *Video 4.1: Social Engineering & Malware*
  - *Video 4.2: Firewalls and IDS/IPS* (Includes Alt-text for network perimeter diagrams)
- **Reading Guide:** Chapter 6-7 Overview.
- **Lab Assignment:** Configuring a basic host-based firewall.
- **Assessment:** Quiz 4 (Domain 4.0) with Distractor Analysis.

### Module 5: Network Troubleshooting (Domain 5.0)
- **Learning Objectives:** Apply the CompTIA troubleshooting methodology.
- **Micro-Learning Video Assets:**
  - *Video 5.1: The 7-Step Troubleshooting Model*
  - *Video 5.2: Command Line Tools (ping, tracert, netstat)*
- **Reading Guide:** Chapter 8 Guide.
- **Lab Assignment:** Troubleshooting scenario in VM environment.
- **Assessment:** Quiz 5 (Domain 5.0) with Distractor Analysis.

---

## Pedagogical Standards Check
- [x] **Micro-Learning:** All videos capped at 5-7 minutes.
- [x] **Question Banks:** All quizzes map to N10-008 domains and use Distractor Analysis.
- [x] **Accessibility:** Visual aids in videos require explicitly written Alt-text scripts.
- [x] **Labs:** Tuned for local execution via VirtualBox/Linux VMs.
