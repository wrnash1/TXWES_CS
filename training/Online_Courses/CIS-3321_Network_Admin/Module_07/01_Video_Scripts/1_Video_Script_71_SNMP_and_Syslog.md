### 1. Video Script 7.1: SNMP and Syslog
**Audio:** "To monitor 100 switches, we use the Simple Network Management Protocol (SNMP). SNMP uses a manager to pull metrics from agents using MIBs (Management Information Bases). Always use SNMPv3 because it encrypts the data. Syslog, on the other hand, is how devices push their event logs to a central server for analysis."
