### 4. Quiz 7 (Question Bank)
**Question 1:** Why is SNMPv3 highly recommended over SNMPv2?
A) It uses TCP instead of UDP.
B) It supports IPv6.
C) It provides authentication and encryption.
D) It compresses data for faster polling.
*   **Correct:** C) It provides authentication and encryption.
*   **Distractor Analysis:** SNMPv1 and v2 send community strings (passwords) in cleartext, making them massive security risks. SNMPv3 encrypts this traffic.
