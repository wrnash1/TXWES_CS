### 1. Video Script 6.1: Wireless Security (WPA2 vs WPA3)
**Audio:** "WEP and WPA are dead. Today, we use WPA2 and WPA3. WPA2 uses AES encryption, but it is vulnerable to offline dictionary attacks if you capture the 4-way handshake. WPA3 fixes this by using Simultaneous Authentication of Equals (SAE), making offline cracking virtually impossible."
