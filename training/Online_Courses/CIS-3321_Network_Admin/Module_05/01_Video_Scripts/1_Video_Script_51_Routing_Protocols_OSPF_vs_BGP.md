### 1. Video Script 5.1: Routing Protocols (OSPF vs. BGP)
**Audio:** "Welcome to Module 5. Static routing works for small networks, but the internet uses dynamic routing protocols. OSPF is an Interior Gateway Protocol used *inside* a company, calculating paths based on bandwidth cost. BGP is the Exterior Gateway Protocol that runs the actual internet, making decisions based on network policies."
**Visual:** Diagram showing a company network (OSPF) connecting to the global Internet (BGP).
