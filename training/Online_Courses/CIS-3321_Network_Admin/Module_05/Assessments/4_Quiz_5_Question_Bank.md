### 4. Quiz 5 (Question Bank)
**Question 1:** Which protocol is used to prevent broadcast storms caused by switching loops?
A) OSPF   B) BGP   C) STP   D) ARP
*   **Correct:** C) STP (Spanning Tree Protocol)
*   **Distractor Analysis:** OSPF and BGP are Layer 3 routing protocols, not Layer 2 loop prevention protocols. ARP resolves IP to MAC.
