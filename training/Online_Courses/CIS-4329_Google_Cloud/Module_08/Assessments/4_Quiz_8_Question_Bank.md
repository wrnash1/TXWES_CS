### 4. Quiz 8 (Question Bank)
**Question 1:** Which command-line tool is specifically used to create, manage, and copy objects into Google Cloud Storage buckets?
A) gcloud
B) bq
C) gsutil
D) kubectl
*   **Correct:** C) gsutil
*   **Distractor Analysis:** `gcloud` is for core compute/networking. `bq` is for BigQuery. `kubectl` is for Kubernetes. `gsutil` is the dedicated utility for Cloud Storage.
