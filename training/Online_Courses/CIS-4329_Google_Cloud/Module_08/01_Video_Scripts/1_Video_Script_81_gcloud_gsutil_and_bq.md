### 1. Video Script 8.1: gcloud, gsutil, and bq
**Audio:** "The ACE exam requires you to be comfortable in the Cloud Shell. You must know the difference between the three main command-line tools. `gcloud` manages core infrastructure like VMs, networks, and IAM. `gsutil` manages Cloud Storage buckets and objects. `bq` manages BigQuery datasets and tables. If the exam asks you how to copy a file to a bucket, look for the answer starting with `gsutil cp`."
