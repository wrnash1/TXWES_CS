### 4. Quiz 7 (Question Bank)
**Question 1:** Your company wants to perform complex, historical analysis on its Google Cloud spending using SQL queries. Which feature should you configure?
A) Cloud Billing Budgets and Alerts
B) Pricing Calculator
C) Billing Export to BigQuery
D) Cloud Audit Logs
*   **Correct:** C) Billing Export to BigQuery
*   **Distractor Analysis:** Budgets only send alerts; they do not store historical data for SQL querying. BigQuery is explicitly designed for querying massive datasets like detailed billing logs.
