### 2. Video Script 2.2: Cloud Storage Classes
**Estimated Duration:** 5 minutes
**Visual:** A table comparing Standard, Nearline, Coldline, and Archive storage classes.
**[Alt-text: A four-column table showing Cloud Storage classes. Standard: Frequent access, highest storage cost, lowest access cost. Nearline: Access < 1/month. Coldline: Access < 1/quarter. Archive: Access < 1/year, lowest storage cost, highest access cost.]**
**Audio:** "For unstructured data like images, backups, or videos, we use Cloud Storage. Cloud Storage is object storage, meaning files are stored in 'Buckets' instead of a traditional file system. To save money, Google offers different storage classes.
Use **Standard** for files you access daily, like website images. 
Use **Nearline** for data accessed once a month.
Use **Coldline** for data accessed once a quarter, like disaster recovery testing.
Use **Archive** for data accessed once a year, like regulatory compliance backups. Remember, as the storage gets cheaper (Archive), the cost to retrieve the data gets much higher."
