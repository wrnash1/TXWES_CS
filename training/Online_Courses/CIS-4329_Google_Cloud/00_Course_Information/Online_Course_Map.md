# Online Course Map: CIS 4329 - Google Cloud Administration
**Target Certification:** Google Cloud Associate Cloud Engineer (ACE)

## Overview
This online course map outlines a new course designed for the Google Cloud Associate Cloud Engineer certification. The asynchronous design relies on Google Cloud Skills Boost / Qwiklabs for hands-on environments, micro-learning video bursts, and distractor-analyzed question banks for certification readiness.

---

## Module Architecture

### Module 1: Setting up a Cloud Solution Environment (Domain 1)
- **Learning Objectives:** Set up cloud projects, manage billing configuration, and install/configure the CLI.
- **Micro-Learning Video Assets (5-7 mins):**
  - *Video 1.1: The Google Cloud Resource Hierarchy* (Includes Alt-text for Organization/Folder/Project diagrams)
  - *Video 1.2: Cloud Shell vs. Cloud SDK*
- **Reading Guide:** Google Cloud ACE Study Guide - Domain 1.
- **Lab Assignment:** Initialize a GCP project and configure gcloud CLI locally.
- **Assessment:** Quiz 1 (Domain 1) with Distractor Analysis.

### Module 2: Planning and Configuring a Cloud Solution (Domain 2)
- **Learning Objectives:** Plan compute, storage, and network resources.
- **Micro-Learning Video Assets:**
  - *Video 2.1: Compute Options (GCE, GKE, App Engine, Cloud Run)*
  - *Video 2.2: Cloud Storage Classes and Lifecycle Policies* (Includes Alt-text for storage class comparison charts)
- **Reading Guide:** Google Cloud ACE Study Guide - Domain 2.
- **Lab Assignment:** Design a VPC network and estimate costs using the Pricing Calculator.
- **Assessment:** Quiz 2 (Domain 2) with Distractor Analysis.

### Module 3: Deploying and Implementing a Cloud Solution (Domain 3)
- **Learning Objectives:** Deploy Compute Engine instances, App Engine, and Kubernetes clusters.
- **Micro-Learning Video Assets:**
  - *Video 3.1: Provisioning Compute Engine VMs*
  - *Video 3.2: Introduction to Google Kubernetes Engine (GKE)* (Includes Alt-text for Pod/Node architecture)
- **Reading Guide:** Google Cloud ACE Study Guide - Domain 3.
- **Lab Assignment:** Deploy a web application to a GCE instance using startup scripts.
- **Assessment:** Quiz 3 (Domain 3) with Distractor Analysis.

### Module 4: Ensuring Successful Operation of a Cloud Solution (Domain 4)
- **Learning Objectives:** Manage Compute Engine resources, monitor with Cloud Operations (formerly Stackdriver).
- **Micro-Learning Video Assets:**
  - *Video 4.1: Managed Instance Groups and Autoscaling*
  - *Video 4.2: Cloud Monitoring and Logging* (Includes Alt-text for Cloud Monitoring dashboards)
- **Reading Guide:** Google Cloud ACE Study Guide - Domain 4.
- **Lab Assignment:** Configure a load balancer and stress test to trigger autoscaling.
- **Assessment:** Quiz 4 (Domain 4) with Distractor Analysis.

### Module 5: Configuring Access and Security (Domain 5)
- **Learning Objectives:** Manage Identity and Access Management (IAM), secure service accounts.
- **Micro-Learning Video Assets:**
  - *Video 5.1: IAM Roles (Primitive, Predefined, Custom)*
  - *Video 5.2: Service Accounts and Principle of Least Privilege*
- **Reading Guide:** Google Cloud ACE Study Guide - Domain 5.
- **Lab Assignment:** Assign IAM roles and audit access using Cloud Audit Logs.
- **Assessment:** Quiz 5 (Domain 5) with Distractor Analysis.

---

## Pedagogical Standards Check
- [x] **Micro-Learning:** All videos capped at 5-7 minutes.
- [x] **Question Banks:** All quizzes map to ACE domains and use Distractor Analysis.
- [x] **Accessibility:** Visual aids in videos require explicitly written Alt-text scripts.
- [x] **Labs:** Tuned for Google Cloud Student Credits / Qwiklabs.
