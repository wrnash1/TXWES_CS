## 3. Updated Online Syllabus
**Course:** CIS 4329 - Google Cloud Administration
**Instructor:** Professor Nash
**Format:** 100% Online Asynchronous
**Target Certification:** Google Cloud Associate Cloud Engineer (ACE)

**Course Description:**
An introduction to deploying applications, monitoring operations, and managing enterprise solutions on Google Cloud Platform. The course covers the GCP resource hierarchy, billing, compute options (VMs, Containers, Serverless), storage, and networking.

**Required Materials:**
1. **Zero Textbook Cost (ZTC):** All required reading materials, official Google Cloud documentation, and study guides are provided completely free within the Blackboard course modules or via the `ZTC_OER_Reading_Materials.md` document. No textbook purchase is required.
2. Active Google Cloud Student Account (Credits provided by the university).

**Course Objectives:**
By the end of this course, students will be able to:
1. Set up a cloud solution environment, including projects, billing, and IAM.
2. Plan and configure cloud solutions using the appropriate compute, storage, and database services.
3. Deploy and implement compute resources via the GCP Console and `gcloud` CLI.
4. Ensure successful operation of a cloud solution using Autoscaling, Load Balancing, and Cloud Monitoring.
