### 2. Video Script 1.2: Cloud Shell vs. Cloud SDK
**Estimated Duration:** 5 minutes
**Visual:** Split screen. Left: Google Cloud Console in a web browser with Cloud Shell open at the bottom. Right: A local Mac terminal running gcloud.
**[Alt-text: Left side shows a web browser logged into Google Cloud with the integrated Cloud Shell terminal open at the bottom. Right side shows a standalone terminal app on a local computer executing a 'gcloud compute instances list' command.]**
**Audio:** "To manage GCP, you use the `gcloud` command-line tool. You have two ways to access it. You can install the Cloud SDK directly on your local laptop. Or, you can use Cloud Shell. Cloud Shell is a free, temporary Linux virtual machine provided right inside your web browser. It comes with `gcloud`, Docker, and Terraform pre-installed, and it is automatically authenticated to your account. For the ACE exam, remember that Cloud Shell is ephemeral—it deletes itself if inactive, but it gives you a persistent 5GB home directory."
