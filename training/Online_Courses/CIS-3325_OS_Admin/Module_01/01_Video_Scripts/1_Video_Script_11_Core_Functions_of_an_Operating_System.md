### 1. Video Script 1.1: Core Functions of an Operating System
**Estimated Duration:** 6 minutes
**Visual:** Instructor on camera.
**Audio:** "Welcome to CIS 3325, Operating System Administration. Before we dive into Linux commands and Windows Active Directory, we need to understand what an OS actually does. At its core, an OS is the bridge between you, your applications, and the physical hardware."
**Visual:** Graphic showing a sandwich model: User -> Application -> Operating System -> Hardware.
**[Alt-text: A vertical block diagram showing four layers. Top layer: User. Second layer: Application. Third layer: Operating System. Bottom layer: Hardware (CPU, RAM, Disk). Arrows point down indicating control flow.]**
**Audio:** "The OS manages memory, processes, file storage, and hardware devices. Without an OS, every single program you write would need to know exactly how to spin a hard drive or send voltage to a monitor. In this course, we will focus heavily on Server Operating Systems, which differ from Desktop OSs by prioritizing background services, networking, and high availability over graphical user interfaces."
