### 4. Lab 1: Installing Ubuntu Server and Windows Server
**Objective:** Deploy two VMs using VirtualBox.
**Instructions:**
1. Download the Ubuntu 22.04 LTS Server ISO and the Windows Server 2022 Evaluation ISO.
2. In VirtualBox, click **New**.
3. Name: `Ubuntu_Server_1`, Type: `Linux`, Version: `Ubuntu (64-bit)`.
4. Allocate 2048 MB RAM and a 25 GB dynamically allocated Virtual Hard Disk.
5. Mount the Ubuntu ISO in the VM's optical drive settings and start the VM. Follow the installation prompts.
6. Repeat the process for Windows Server, allocating at least 4096 MB RAM and a 50 GB disk.
**Deliverable:** Take a screenshot of both VMs powered on and sitting at their respective login screens within VirtualBox.
