### 3. Reading Guide: OS Architecture
**High-Yield Concepts:**
1. **Type 1 vs. Type 2 Hypervisors:** Type 1 (Bare Metal) runs directly on the hardware (e.g., VMware ESXi, Hyper-V). Type 2 runs on a host OS (e.g., VirtualBox, VMware Workstation).
2. **Kernel vs. User Space:** The kernel is the core of the OS with complete control over everything in the system. User space is where normal applications run, restricted from direct hardware access to prevent crashes.
3. **Open Source vs. Closed Source:** Linux is open-source (code is freely available). Windows Server is closed-source/proprietary.
