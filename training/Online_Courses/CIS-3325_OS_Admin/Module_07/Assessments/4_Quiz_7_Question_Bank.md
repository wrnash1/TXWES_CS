### 4. Quiz 7 (Question Bank)
**Question 1:** Which command searches for a specific string of text within a file?
A) find   B) locate   C) grep   D) cat
*   **Correct:** C) grep
*   **Distractor Analysis:** `find` and `locate` search for file *names*, not the contents inside the files. `cat` prints the entire file to the screen without filtering.
