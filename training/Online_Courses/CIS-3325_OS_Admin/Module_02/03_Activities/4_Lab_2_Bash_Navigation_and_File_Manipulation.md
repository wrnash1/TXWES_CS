### 4. Lab 2: Bash Navigation and File Manipulation
**Objective:** Navigate the FHS and manipulate files using only the command line.
**Instructions:**
1. Boot your Ubuntu Server VM and log in.
2. Determine your current location: `pwd`
3. Navigate to the system configuration directory: `cd /etc`
4. List all files in long format, including hidden files: `ls -la`
5. Navigate to your home directory using the tilde shortcut: `cd ~`
6. Create a new directory named 'test_lab': `mkdir test_lab`
7. Create an empty file inside it: `touch test_lab/empty.txt`
**Deliverable:** Take a screenshot of your terminal showing the successful execution of the `mkdir` and `touch` commands followed by an `ls test_lab` verifying the file was created.
