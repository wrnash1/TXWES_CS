### 1. Video Script 2.1: The Linux Philosophy and Open Source
**Estimated Duration:** 5 minutes
**Visual:** Instructor on camera.
**Audio:** "Welcome to Module 2. Windows dominates the corporate desktop, but Linux dominates the cloud, web servers, and mobile devices. Why? Because of the open-source philosophy. In the 1990s, Linus Torvalds created the Linux kernel and released it for free. Anyone can view, modify, and distribute the source code."
**Visual:** A diagram showing the Linux Kernel in the center, surrounded by GNU tools (bash, ls, cp), surrounded by Distributions (Ubuntu, RedHat, Alpine).
**[Alt-text: Three concentric circles. The center circle is labeled 'Linux Kernel'. The middle ring is labeled 'GNU Utilities'. The outer ring is labeled 'Linux Distributions'.]**
**Audio:** "When we say 'Linux', we technically just mean the kernel. The commands you type, like `ls` or `bash`, come from the GNU project. When you package the kernel, the GNU tools, and a package manager together, you get a 'Distribution' or 'Distro' like Ubuntu or RedHat."
