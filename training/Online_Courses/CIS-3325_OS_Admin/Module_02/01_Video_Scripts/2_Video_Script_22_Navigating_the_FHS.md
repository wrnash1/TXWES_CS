### 2. Video Script 2.2: Navigating the FHS
**Estimated Duration:** 7 minutes
**Visual:** Screencast of a Linux terminal.
**[Alt-text: A screencast of a Linux terminal window. The instructor types 'cd /' then 'ls -l' to show the root directory structure including /etc, /var, and /home.]**
**Audio:** "Unlike Windows, which uses drive letters like C: and D:, Linux uses a single unified tree structure called the Filesystem Hierarchy Standard, or FHS. Everything starts at the root, represented by a forward slash `/`. 
If you want to configure a system service, you go to `/etc`. 
If you want to see log files that change constantly, you go to `/var`. 
If you want to save your personal documents, you go to `/home/yourusername`.
Let's practice navigating this using the `cd` (change directory), `pwd` (print working directory), and `ls` (list) commands."
