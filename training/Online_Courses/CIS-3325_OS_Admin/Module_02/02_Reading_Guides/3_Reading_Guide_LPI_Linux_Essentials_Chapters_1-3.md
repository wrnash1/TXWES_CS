### 3. Reading Guide: LPI Linux Essentials Chapters 1-3
**High-Yield Concepts:**
1. **Absolute vs. Relative Paths:** An absolute path always starts at the root (e.g., `/var/log/syslog`). A relative path starts from your current directory (e.g., `../log/syslog`).
2. **Command Syntax:** The standard format is `Command [Options] [Arguments]`. For example: `ls -l /etc`. `ls` is the command, `-l` is the option (long format), and `/etc` is the argument.
3. **Getting Help:** Use the `man` command to read the manual page for any other command (e.g., `man ls`).
