# CIS 3325: Zero Textbook Cost (ZTC) OER Guide
**Target Certification:** LPI Linux Essentials (010-160)

This course uses a Zero Textbook Cost model. Instead of purchasing a textbook, you will use the following free Open Educational Resources (OER) curated directly from the Linux Professional Institute (LPI) and official documentation.

## Primary OER Textbook Replacements
You will rely heavily on the official, free learning materials provided directly by the certification vendor.

### Module 1-4: The Command Line & Basic Administration
*   **Reading 1:** [Official LPI Linux Essentials Free Learning Materials](https://learning.lpi.org/en/learning-materials/010-160/)
    *   *Focus:* Complete Topic 1 (The Linux Community) through Topic 3 (The Power of the Command Line).
*   **Reading 2:** The Linux `man` Pages
    *   *Focus:* Instead of a textbook, your terminal is your book. For any command discussed in the modules, simply type `man [command]` (e.g., `man ls`) to read the official manual.

### Module 5-8: Scripting, Networking, & Hardware
*   **Reading 3:** [Official LPI Linux Essentials Free Learning Materials](https://learning.lpi.org/en/learning-materials/010-160/)
    *   *Focus:* Complete Topic 4 (The Linux Operating System) and Topic 5 (Security and File Permissions).
*   **Reading 4:** [Bash Scripting Tutorial for Beginners (FreeCodeCamp)](https://www.freecodecamp.org/news/bash-scripting-tutorial-linux-shell-script-and-command-line-for-beginners/)

### Module 9-16: Advanced Text Processing & Operations
*   **Reading 5:** [Linux Journey (Free Interactive Course)](https://linuxjourney.com/)
    *   *Focus:* Read the sections on "Text-Fu" (for `grep`, `sed`, `awk`) and "Network Configuration".
*   **Reading 6:** [Official LPI 010-160 Exam Objectives](https://www.lpi.org/our-certifications/linux-essentials-overview)
    *   *Focus:* Review the official weighting of the exam domains.

> **Instructor Note:** The LPI provides an incredibly comprehensive, completely free digital textbook linked above. This is the only text you will need to pass the certification.
