# Online Course Map: CIS 3325 - Operating System Administration
**Target Certification:** LPI Linux Essentials (010-160) & Windows Server Foundations

## Overview
This online course map outlines the modular transition of CIS 3325 to an asynchronous format. It focuses heavily on hands-on local virtual labs (VirtualBox) running Linux VMs, supported by micro-learning videos and robust distractor analysis for certification readiness.

---

## Module Architecture

### Module 1: OS Fundamentals & Windows Basics
- **Learning Objectives:** Compare Windows and Linux architectures, install OS in VirtualBox.
- **Micro-Learning Video Assets (5-7 mins):**
  - *Video 1.1: Core Functions of an Operating System*
  - *Video 1.2: Hypervisors and VirtualBox Setup* (Includes Alt-text for VM configuration screens)
- **Reading Guide:** MS Learn OS Architecture Overview.
- **Lab Assignment:** Install Ubuntu Server and Windows Server Evaluation in VirtualBox.
- **Assessment:** Quiz 1 with Distractor Analysis.

### Module 2: Introduction to Linux & Command Line
- **Learning Objectives:** Navigate the Linux filesystem, execute basic bash commands.
- **Micro-Learning Video Assets:**
  - *Video 2.1: The Linux Philosophy and Open Source*
  - *Video 2.2: Navigating the FHS (Filesystem Hierarchy Standard)*
- **Reading Guide:** LPI Linux Essentials Chapters 1-3.
- **Lab Assignment:** Bash navigation and file manipulation exercises.
- **Assessment:** Quiz 2 (LPI Domain 2/3) with Distractor Analysis.

### Module 3: Users, Groups, and Permissions (Linux & Windows)
- **Learning Objectives:** Manage identities, configure NTFS and Ext4 permissions.
- **Micro-Learning Video Assets:**
  - *Video 3.1: Linux Ownership (chmod, chown)* (Includes Alt-text for permission matrices)
  - *Video 3.2: Active Directory vs. Local Linux Users*
- **Reading Guide:** LPI Linux Essentials Chapter 5.
- **Lab Assignment:** Creating users and assigning group permissions.
- **Assessment:** Quiz 3 with Distractor Analysis.

### Module 4: Processes and Service Management
- **Learning Objectives:** Monitor system resources, manage systemd / Windows Services.
- **Micro-Learning Video Assets:**
  - *Video 4.1: The 'ps', 'top', and 'kill' commands*
  - *Video 4.2: Introduction to systemctl*
- **Reading Guide:** LPI Linux Essentials Chapters 6-7.
- **Lab Assignment:** Service management and resource monitoring.
- **Assessment:** Quiz 4 with Distractor Analysis.

### Module 5: Scripting and Automation
- **Learning Objectives:** Write basic bash scripts to automate administration tasks.
- **Micro-Learning Video Assets:**
  - *Video 5.1: Bash Scripting Basics (Variables & Loops)*
  - *Video 5.2: Automating with Cron* (Includes Alt-text for crontab syntax diagrams)
- **Reading Guide:** LPI Linux Essentials Chapter 10.
- **Lab Assignment:** Create an automated backup bash script.
- **Assessment:** Quiz 5 with Distractor Analysis.

---

- [x] **Micro-Learning:** All videos capped at 5-7 minutes.
- [x] **Question Banks:** All quizzes map to LPI 010-160 domains and use Distractor Analysis.
