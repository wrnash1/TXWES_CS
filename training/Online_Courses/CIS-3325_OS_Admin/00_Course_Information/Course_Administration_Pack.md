## 3. Updated Online Syllabus
**Course:** CIS 3325 - Operating System Administration
**Instructor:** Professor Nash
**Format:** 100% Online Asynchronous
**Target Certification:** LPI Linux Essentials (010-160)

**Course Description:**
An introduction to the fundamental concepts of operating systems, with a heavy emphasis on Linux administration. Topics include file system navigation, user management, permissions, process control, and basic shell scripting.

**Required Materials:**
1. **Zero Textbook Cost (ZTC):** All required reading materials, official documentation, and study guides are provided completely free within the Blackboard course modules or via the `ZTC_OER_Reading_Materials.md` document. No textbook purchase is required.
2. Oracle VirtualBox.
3. Ubuntu Server 22.04 LTS ISO (Free download).

**Course Objectives:**
By the end of this course, students will be able to:
1. Understand the Linux and open-source industry and knowledge of the most popular open-source Applications.
2. Understand the major components of the Linux operating system, and have the technical proficiency to navigate and work at the Linux command line.
3. Understand basic security and administration-related topics such as user/group management, working on the command line, and permissions.

## 4. Grading Policy & Rubrics
**Grade Breakdown:**
*   **Weekly Lab Activities (CLI Execution):** 40%
*   **Weekly Quizzes:** 30%
*   **Final Certification Exam:** 30%

**Standard Lab Grading Rubric (100 Points):**
*   **Task Completion (50 pts):** All commands were executed successfully.
*   **Deliverable/Screenshot (30 pts):** Screenshot clearly shows the terminal output proving the command worked (e.g., `ls -l` showing correct permissions).
*   **Troubleshooting/Logic (20 pts):** Clear understanding of the command syntax is demonstrated.
