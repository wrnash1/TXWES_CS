### 3. Reading Guide: LPI Linux Essentials Chapters 6 & 7
**High-Yield Concepts:**
1. **Background vs. Foreground Processes:** Appending an ampersand `&` to the end of a command (e.g., `sleep 100 &`) runs the process in the background, freeing up your terminal prompt immediately.
2. **Standard Streams:** 
   * *stdin (0):* Standard Input (usually the keyboard).
   * *stdout (1):* Standard Output (normal output printed to the screen).
   * *stderr (2):* Standard Error (error messages).
3. **Redirection:** Use `>` to redirect stdout to a file (overwriting it). Use `>>` to append to a file. Use `2>` to redirect errors to a file.
