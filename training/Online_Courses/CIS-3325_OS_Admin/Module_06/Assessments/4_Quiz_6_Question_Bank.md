### 4. Quiz 6 (Question Bank)
**Question 1:** Which modern Linux command has replaced `ifconfig` for displaying IP addresses on network interfaces?
A) netstat
B) ping
C) ip addr
D) route
*   **Correct:** C) ip addr
*   **Distractor Analysis:** `netstat` shows open ports. `ping` tests connectivity. `route` shows routing tables (and is also largely replaced by `ip route`).
