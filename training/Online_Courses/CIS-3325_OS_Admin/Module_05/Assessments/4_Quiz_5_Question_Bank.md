### 4. Quiz 5 (Question Bank)
**Question 1:** Which command must be executed on a newly created text file containing bash commands before you can run it directly?
A) chown root script.sh
B) chmod +x script.sh
C) systemctl start script.sh
D) bash enable script.sh
*   **Correct:** B) chmod +x script.sh
*   **Distractor Analysis:** A script needs the 'execute' (x) permission to run. `chown` changes ownership, and `systemctl` manages services, not standard scripts.
