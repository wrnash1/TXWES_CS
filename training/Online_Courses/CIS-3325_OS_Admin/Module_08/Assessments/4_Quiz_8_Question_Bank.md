### 4. Quiz 8 (Question Bank)
**Question 1:** Which directory contains files that represent physical hardware devices attached to the system?
A) /etc   B) /var   C) /dev   D) /bin
*   **Correct:** C) /dev
*   **Distractor Analysis:** In Linux, "everything is a file." Hardware devices (like hard drives or terminals) are mapped to files inside the `/dev` (device) directory.
