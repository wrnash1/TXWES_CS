### 4. Quiz 6 (Question Bank)
**Question 1:** In IIS, what is the purpose of placing different websites into separate Application Pools?
A) To assign different IP addresses to each site.
B) To isolate worker processes so that a crash in one site does not affect the others.
C) To encrypt the web traffic using SSL/TLS.
D) To balance the network load across multiple servers.
*   **Correct:** B) To isolate worker processes.
*   **Distractor Analysis:** Bindings handle IPs and ports. Certificates handle SSL encryption. Network Load Balancers (NLB) handle load balancing. Application pools provide process isolation.
