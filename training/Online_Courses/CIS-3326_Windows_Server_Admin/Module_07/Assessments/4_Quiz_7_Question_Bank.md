### 4. Quiz 7 (Question Bank)
**Question 1:** Which Remote Desktop Services role service is explicitly designed to allow external users to access internal RDS resources securely over the internet without using a traditional VPN?
A) RD Session Host
B) RD Connection Broker
C) RD Gateway
D) RD Licensing
*   **Correct:** C) RD Gateway
*   **Distractor Analysis:** The Session Host actually runs the apps. The Broker manages connections. The Gateway encapsulates RDP traffic inside an HTTPS tunnel for secure internet traversal.
