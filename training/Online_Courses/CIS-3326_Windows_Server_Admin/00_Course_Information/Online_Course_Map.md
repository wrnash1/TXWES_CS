# Online Course Map: CIS 3326 - Windows Server Administration
**Target Certification:** Microsoft Windows Server Administration

## Overview
This online course map translates CIS 3326 into an asynchronous format. The course relies on Microsoft Evaluation VMs running locally to simulate enterprise environments, utilizing 5-7 minute video modules and certification-aligned question banks.

---

## Module Architecture

### Module 1: Windows Server Installation and Configuration
- **Learning Objectives:** Install Windows Server 2022/2019, configure local settings.
- **Micro-Learning Video Assets (5-7 mins):**
  - *Video 1.1: Server Core vs. Desktop Experience*
  - *Video 1.2: Post-Installation Configuration* (Includes Alt-text for Server Manager interface)
- **Reading Guide:** Microsoft Press Chapter 1.
- **Lab Assignment:** Install Windows Server VM, configure static IP and hostname.
- **Assessment:** Quiz 1 with Distractor Analysis.

### Module 2: Active Directory Domain Services (AD DS)
- **Learning Objectives:** Promote a server to a Domain Controller, manage AD objects.
- **Micro-Learning Video Assets:**
  - *Video 2.1: The Logical Structure of AD DS* (Includes Alt-text for Forest/Domain diagrams)
  - *Video 2.2: Creating Users, Computers, and OUs*
- **Reading Guide:** Microsoft Press Chapter 2.
- **Lab Assignment:** Install AD DS role and promote to Domain Controller.
- **Assessment:** Quiz 2 with Distractor Analysis.

### Module 3: Group Policy Objects (GPO)
- **Learning Objectives:** Create, link, and troubleshoot Group Policies.
- **Micro-Learning Video Assets:**
  - *Video 3.1: GPO Hierarchy and Inheritance*
  - *Video 3.2: Security Filtering and WMI Filters*
- **Reading Guide:** Microsoft Press Chapter 3.
- **Lab Assignment:** Enforce password policies and map drives via GPO.
- **Assessment:** Quiz 3 with Distractor Analysis.

### Module 4: File Services and Storage
- **Learning Objectives:** Configure NTFS, ReFS, and network shares.
- **Micro-Learning Video Assets:**
  - *Video 4.1: NTFS Permissions vs. Share Permissions*
  - *Video 4.2: Storage Spaces and RAID levels* (Includes Alt-text for RAID architecture)
- **Reading Guide:** Microsoft Press Chapter 4.
- **Lab Assignment:** Configure a DFS Namespace and File Server Resource Manager (FSRM).
- **Assessment:** Quiz 4 with Distractor Analysis.

### Module 5: Server Security and Monitoring
- **Learning Objectives:** Implement Windows Defender, use Performance Monitor.
- **Micro-Learning Video Assets:**
  - *Video 5.1: Windows Firewall with Advanced Security*
  - *Video 5.2: Event Viewer and Performance Monitor*
- **Reading Guide:** Microsoft Press Chapter 5.
- **Lab Assignment:** Create data collector sets and configure firewall rules.
- **Assessment:** Quiz 5 with Distractor Analysis.

---

## Pedagogical Standards Check
- [x] **Micro-Learning:** All videos capped at 5-7 minutes.
- [x] **Question Banks:** All quizzes map to Microsoft objectives and use Distractor Analysis.
- [x] **Accessibility:** Visual aids in videos require explicitly written Alt-text scripts.
- [x] **Labs:** Tuned for local execution via VirtualBox.
