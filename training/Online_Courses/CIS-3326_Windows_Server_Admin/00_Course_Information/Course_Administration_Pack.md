## 3. Updated Online Syllabus
**Course:** CIS 3326 - Windows Server Administration
**Instructor:** Professor Nash
**Format:** 100% Online Asynchronous
**Target Certification:** Microsoft Windows Server Administration

**Course Description:**
This course covers the installation, configuration, and management of Windows Server environments. Core topics include Active Directory Domain Services (AD DS), Group Policy Objects (GPOs), file and print services, and Server Core management via PowerShell.

**Required Materials:**
1. **Zero Textbook Cost (ZTC):** All required reading materials, Microsoft Learn paths, and study guides are provided completely free within the Blackboard course modules or via the `ZTC_OER_Reading_Materials.md` document. No textbook purchase is required.
2. Oracle VirtualBox.
3. Windows Server 2022 Evaluation ISO (Free download from Microsoft).

**Course Objectives:**
By the end of this course, students will be able to:
1. Deploy and configure Windows Server Core and Desktop Experience.
2. Install and manage Active Directory Domain Services and establish a domain hierarchy.
3. Create and link Group Policy Objects to manage users and computers centrally.
4. Configure file services, DFS namespaces, and restrictive NTFS permissions.

## 4. Grading Policy & Rubrics
**Grade Breakdown:**
*   **Weekly Lab Activities (VM Deployments):** 40%
*   **Weekly Quizzes:** 30%
*   **Final Certification Exam:** 30%

**Standard Lab Grading Rubric (100 Points):**
*   **Task Completion (50 pts):** The server role/feature was successfully installed and configured.
*   **Deliverable/Screenshot (30 pts):** Screenshot shows the Server Manager or PowerShell output confirming the configuration (e.g., ADUC showing the new OU).
*   **Troubleshooting/Logic (20 pts):** Demonstration of understanding administrative concepts.
