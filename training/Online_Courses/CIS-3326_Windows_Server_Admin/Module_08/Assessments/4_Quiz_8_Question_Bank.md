### 4. Quiz 8 (Question Bank)
**Question 1:** Which type of backup is specifically required to successfully back up and restore the Active Directory database on a Domain Controller?
A) Bare Metal Recovery
B) System State Backup
C) Incremental File Backup
D) Hyper-V Snapshot
*   **Correct:** B) System State Backup
*   **Distractor Analysis:** System State explicitly targets critical OS components, the registry, and the AD database. File backups and hypervisor snapshots are not supported methods for restoring Active Directory consistently.
