### 2. Video Script 4.2: Storage Spaces and RAID levels
**Estimated Duration:** 7 minutes
**Visual:** Diagrams showing RAID 0, RAID 1, and RAID 5.
**[Alt-text: Three diagrams. RAID 0 (Striping): Data split across two disks, no redundancy. RAID 1 (Mirroring): Identical data copied to two disks. RAID 5 (Parity): Data and parity bits striped across three or more disks.]**
**Audio:** "Hard drives fail. It's a certainty. To protect against this, we use RAID (Redundant Array of Independent Disks). RAID 1 mirrors data perfectly across two drives. If one dies, you lose nothing. RAID 5 stripes data across three or more drives, using parity math to recreate data if one drive fails. Windows Server offers a software-defined storage feature called 'Storage Spaces'. Instead of relying on expensive hardware RAID controllers, Storage Spaces allows you to group cheap, commodity hard drives into a 'Storage Pool' and create virtual disks with software-defined mirroring or parity."
