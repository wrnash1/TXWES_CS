### 2. Video Script 2.2: Creating Users, Computers, and OUs
**Estimated Duration:** 7 minutes
**Visual:** Screencast of Active Directory Users and Computers (ADUC) MMC snap-in on Windows Server.
**[Alt-text: A screencast of the Windows Server desktop running the 'Active Directory Users and Computers' tool. The user right-clicks the domain name, selects New -> Organizational Unit, and names it 'Marketing'.]**
**Audio:** "Let's see this in action. A server that holds a copy of the Active Directory database is called a Domain Controller. Once promoted, we use the tool 'Active Directory Users and Computers'—often called ADUC. 
Watch as I right-click the domain and create a new OU called 'Marketing'. By creating an OU, I can isolate Marketing's computers and apply specific security policies to them later. Now, inside the Marketing OU, I right-click and create a new User object. I give them a login name and a temporary password. This user can now log into ANY computer that has been joined to this domain."
