### 3. Reading Guide: Microsoft Press Chapter 2
**High-Yield Concepts:**
1. **Domain Controller Promotion:** Installing the AD DS role is not enough; you must then run the wizard to 'Promote this server to a domain controller'.
2. **Global Catalog (GC):** A domain controller that stores a partial, read-only copy of all objects in the entire forest. It's used for forest-wide searches and universal group membership lookups.
3. **FSMO Roles:** Flexible Single Master Operations roles. There are 5 roles (Schema Master, Domain Naming Master, RID Master, PDC Emulator, Infrastructure Master) that handle specific tasks to prevent database conflicts.
