### 1. Video Script 1.1: Server Core vs. Desktop Experience
**Estimated Duration:** 6 minutes
**Visual:** Instructor on camera.
**Audio:** "Welcome to Windows Server Administration. When you install Windows Server 2022, you are immediately faced with a critical choice: Server Core or Desktop Experience. By default, Microsoft pushes you towards Server Core."
**Visual:** Split screen. Left: A black command prompt window (Server Core). Right: The familiar Windows GUI with Server Manager open (Desktop Experience).
**[Alt-text: Split screen comparing Windows Server Core, which shows only a black command prompt, and Windows Server Desktop Experience, which shows a traditional Windows graphical user interface and taskbar.]**
**Audio:** "Server Core has no graphical user interface. You manage it entirely via PowerShell or remotely. Why? It drastically reduces the attack surface, uses far less RAM and CPU, and requires fewer reboots for patches. Desktop Experience gives you the traditional GUI, but at the cost of higher overhead. In modern environments, Core is king."
