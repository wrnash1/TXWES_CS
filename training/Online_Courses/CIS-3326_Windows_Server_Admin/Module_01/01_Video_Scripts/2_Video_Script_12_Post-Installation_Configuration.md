### 2. Video Script 1.2: Post-Installation Configuration
**Estimated Duration:** 5 minutes
**Visual:** Screencast of the 'sconfig' tool in Server Core.
**[Alt-text: A screencast of the 'sconfig' text-based menu in Windows Server Core, showing options to change the computer name, configure network settings, and download updates.]**
**Audio:** "Once your server is booted, it needs an identity. Out of the box, it has a randomly generated name and is looking for a DHCP address. For a server, this is unacceptable. Servers need static IP addresses so clients can reliably find them. If you are on Server Core, you use the `sconfig` utility. This text-based menu allows you to quickly change the hostname, set a static IPv4 address, and enable Remote Desktop."
