## Supplemental Resources

### 5. Python None Type Explained
**Search:** `"Python None NoneType explained" tutorial`
**Why Watch:** `None` confuses many beginners. This type of video clarifies when to use `None`, how it compares to `False` and `0`, and why you should use `is None` rather than `== None`.

### 6. Python Integer Bases — Binary, Octal, Hex
**Search:** `"Python binary octal hexadecimal literals tutorial"`
**Why Watch:** Covers `0b`, `0o`, and `0x` prefixes and why different number bases matter in computing (networking, memory addresses, permissions). Directly supports the PCAP exam content.

### 7. Python Augmented Assignment Operators
**Search:** `"Python augmented assignment operators += -= tutorial"`
**Why Watch:** Short video demonstrating all augmented operators with practical examples. Good practice before the lab.
