## Supplemental

### 4. Python Loop else Clause — Deep Dive
**Search:** `"Python for else while else loop tutorial"`
**Why Watch:** Many Python developers don't know about `for...else`. This video explains when `else` runs and when it's skipped (after `break`), including real-world use cases like prime checkers and search loops.

### 5. Python pass Statement Explained
**Search:** `"Python pass statement when to use tutorial"`
**Why Watch:** Explains the difference between `pass`, `continue`, and a bare `return` — helps students avoid confusing `pass` with `continue`.

### 6. Python Infinite Loops and break — Best Practices
**Search:** `"Python while True break pattern tutorial"`
**Why Watch:** Explains why `while True: ... break` is not a bug but a design pattern. Covers input validation and menu systems.
