# Video Resources: Module 1 — Introduction to Python Programming

**Course:** CIS-1310 Introduction to Python Programming
**Institution:** Texas Wesleyan University
**Module Topic:** Python basics, environment setup, print(), comments, PEP 8

> These are curated external YouTube videos and course resources. You do not need to watch all of them — choose the ones that best match your learning style. All are free.

---

## Primary Video Resources

### 1. Python Tutorial for Beginners — Programming with Mosh
**Channel:** Programming with Mosh
**Search:** `"Python Tutorial for Beginners" Programming with Mosh`
**Direct Link:** [https://www.youtube.com/watch?v=kqtD5dpn9C8](https://www.youtube.com/watch?v=kqtD5dpn9C8)
**Covers for Module 1:** Introduction to Python (0:00–15:00), setting up the environment, your first program
**Why Watch:** Mosh gives a clean, fast-paced overview ideal for beginners. His explanation of how Python works (interpreted vs compiled) directly supports the PCAP Section 1 content.

## Supplemental Video Resources

### 5. Setting Up Google Colab for Python (Beginner Guide)
**Search:** `"How to use Google Colab for Python beginners" 2024`
**Why Watch:** Step-by-step walkthrough of Google Colab — adding cells, running code, saving notebooks to Drive, and sharing files with your instructor.

### 6. PEP 8 — Python Style Guide Explained
**Search:** `"PEP 8 Python style guide tutorial" Corey Schafer`
**Why Watch:** A dedicated explanation of naming conventions, line length, indentation rules, and whitespace guidelines. Directly prepares you for PCAP questions on PEP 8.

### 7. Python print() Function — All Parameters Explained
**Search:** `"Python print function sep end parameters explained"`
**Why Watch:** Deep dive into `sep`, `end`, `file`, and `flush` parameters of `print()`. Good if you want complete mastery of the function for the PCAP exam.
