# Course Administration Pack: CIS-1310_Intro_to_Python
**Target Certification:** PCAP (Certified Associate in Python Programming)

## Course Description
This course covers Introduction to Python Programming, covering data types, control flow, functions, modules, and Object-Oriented Programming (OOP).. It is aligned directly with the PCAP (Certified Associate in Python Programming) exam blueprint.

## Required Materials
1. **Zero Textbook Cost (ZTC):** All required reading materials, official documentation, and study guides are provided completely free within the Blackboard course modules or via the `ZTC_OER_Reading_Materials.md` document. No textbook purchase is required.
2. A computer capable of running Google Colab in a web browser or a local Python IDLE installation.
