### 3. Reading Guide: Security+ SY0-701 Domain 1
**High-Yield Concepts:**
1. **Zero Trust Architecture:** The principle of "never trust, always verify." Every request is verified as if it originated from an open network, regardless of whether the user is inside the corporate office or at home.
2. **Defense in Depth:** Layering multiple security controls so that if one fails, others are there to catch the threat (e.g., using a firewall, AND an antivirus, AND user training).
3. **Control Types by Function:** 
   * *Preventive:* Stops the attack (Firewall).
   * *Detective:* Identifies the attack (Intrusion Detection System).
   * *Corrective:* Fixes the damage (Restoring from Backup).
