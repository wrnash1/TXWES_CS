# Online Course Map: CIS 4328 - Information Systems Security
**Target Certification:** CompTIA Security+ (SY0-701)

## Overview
This online course map outlines the modular transition of CIS 4328 to an asynchronous format. It prepares students for the CompTIA Security+ exam by combining micro-learning videos, rigorous distractor-analyzed quiz banks, and hands-on virtual labs emphasizing vendor-neutral security principles.

---

## Module Architecture

### Module 1: General Security Concepts (Domain 1.0)
- **Learning Objectives:** Compare security controls, understand fundamental security concepts (CIA Triad, Zero Trust).
- **Micro-Learning Video Assets (5-7 mins):**
  - *Video 1.1: Security Controls: Physical, Logical, and Administrative*
  - *Video 1.2: The CIA Triad and Non-Repudiation* (Includes Alt-text for CIA triad diagrams)
- **Reading Guide:** Security+ SY0-701 Study Guide - Domain 1.
- **Lab Assignment:** Configure basic security controls on a local Linux VM.
- **Assessment:** Quiz 1 (Domain 1.0) with Distractor Analysis.

### Module 2: Threats, Vulnerabilities, and Mitigations (Domain 2.0)
- **Learning Objectives:** Analyze common threat actors, vectors, and malware.
- **Micro-Learning Video Assets:**
  - *Video 2.1: Phishing, Vishing, and Social Engineering*
  - *Video 2.2: Malware Types: Ransomware to Rootkits* (Includes Alt-text for malware infection vectors)
- **Reading Guide:** Security+ SY0-701 Study Guide - Domain 2.
- **Lab Assignment:** Perform vulnerability scanning using OpenVAS/Nmap.
- **Assessment:** Quiz 2 (Domain 2.0) with Distractor Analysis.

### Module 3: Security Architecture (Domain 3.0)
- **Learning Objectives:** Implement secure enterprise architecture and cryptography basics.
- **Micro-Learning Video Assets:**
  - *Video 3.1: Network Segmentation and Demilitarized Zones (DMZs)*
  - *Video 3.2: Cryptography: Symmetric vs Asymmetric Encryption* (Includes Alt-text for encryption key exchange flowcharts)
- **Reading Guide:** Security+ SY0-701 Study Guide - Domain 3.
- **Lab Assignment:** Generate and exchange SSH and PGP keys.
- **Assessment:** Quiz 3 (Domain 3.0) with Distractor Analysis.

### Module 4: Security Operations (Domain 4.0)
- **Learning Objectives:** Respond to incidents, analyze monitoring data.
- **Micro-Learning Video Assets:**
  - *Video 4.1: The Incident Response Lifecycle*
  - *Video 4.2: Analyzing Logs with SIEMs* (Includes Alt-text for SIEM dashboard layouts)
- **Reading Guide:** Security+ SY0-701 Study Guide - Domain 4.
- **Lab Assignment:** Analyze mock network traffic and system logs to identify an intrusion.
- **Assessment:** Quiz 4 (Domain 4.0) with Distractor Analysis.

### Module 5: Security Program Management and Oversight (Domain 5.0)
- **Learning Objectives:** Understand risk management, compliance, and policies.
- **Micro-Learning Video Assets:**
  - *Video 5.1: Risk Assessments and Mitigation Strategies*
  - *Video 5.2: Compliance Frameworks (PCI-DSS, HIPAA, GDPR)*
- **Reading Guide:** Security+ SY0-701 Study Guide - Domain 5.
- **Lab Assignment:** Draft an Acceptable Use Policy and an Incident Response Plan.
- **Assessment:** Quiz 5 (Domain 5.0) with Distractor Analysis.

---

## Pedagogical Standards Check
- [x] **Micro-Learning:** All videos capped at 5-7 minutes.
- [x] **Question Banks:** All quizzes map to SY0-701 domains and use Distractor Analysis.
- [x] **Accessibility:** Visual aids in videos require explicitly written Alt-text scripts.
- [x] **Labs:** Tuned for local execution via VirtualBox/Kali Linux VMs.
