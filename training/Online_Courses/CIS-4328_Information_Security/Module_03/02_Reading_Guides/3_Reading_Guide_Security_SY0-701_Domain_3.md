### 3. Reading Guide: Security+ SY0-701 Domain 3
**High-Yield Concepts:**
1. **Hashing:** A one-way mathematical function that creates a fixed-size string from data. Used for verifying Integrity. You cannot "decrypt" a hash. Common algorithms: SHA-256, MD5 (deprecated).
2. **Digital Signatures:** Created by hashing a document and then encrypting that hash with the sender's *Private Key*. This proves non-repudiation (the sender definitely sent it) and integrity (the document didn't change).
3. **Honeypots:** Decoy systems placed in a network specifically to attract hackers, allowing defenders to study their tactics without risking real assets.
