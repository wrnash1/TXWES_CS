### 4. Quiz 5 (Question Bank)
**Question 1:** Which access control model evaluates variables such as the user's current physical location, time of day, and device security posture before granting access?
A) Role-Based Access Control (RBAC)
B) Mandatory Access Control (MAC)
C) Attribute-Based Access Control (ABAC)
D) Discretionary Access Control (DAC)
*   **Correct:** C) Attribute-Based Access Control (ABAC)
*   **Distractor Analysis:** RBAC looks only at group/role membership. MAC is used by the military (Top Secret clearances). ABAC evaluates multiple dynamic attributes (time, location, device state).
