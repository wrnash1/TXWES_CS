### 2. Video Script 4.2: Query Insights and Cloud Monitoring
**Estimated Duration:** 5 minutes
**Visual:** Screencast of the Google Cloud Console showing the Query Insights dashboard for Cloud SQL.
**[Alt-text: A screencast of the Cloud SQL Query Insights dashboard. A graph shows 'Database load by query'. Below the graph is a table listing specific SQL queries and their average execution times.]**
**Audio:** "How do you know if your database is healthy? You monitor it. Google Cloud Monitoring tracks CPU utilization, memory, and disk IOPS. You can create alerting policies to email you if the CPU stays above 90% for 5 minutes. But what if the CPU is high because of a badly written SQL query? For that, we use Query Insights. Query Insights visually identifies exactly which SQL statement is causing performance bottlenecks, allowing you to rewrite the query or add an index to the table without guessing."
