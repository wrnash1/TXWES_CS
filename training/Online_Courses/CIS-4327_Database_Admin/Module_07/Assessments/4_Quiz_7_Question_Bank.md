### 4. Quiz 7 (Question Bank)
**Question 1:** What is the primary benefit of using Terraform (Infrastructure as Code) to provision databases instead of using the Google Cloud Console?
A) Terraform provides better database performance.
B) Terraform allows for version-controlled, repeatable, and automated deployments.
C) Terraform automatically converts NoSQL databases to Relational databases.
D) Terraform eliminates the need to pay for Google Cloud resources.
*   **Correct:** B) Terraform allows for version-controlled, repeatable, and automated deployments.
*   **Distractor Analysis:** IaC does not alter performance or billing; it simply automates the API calls to provision the infrastructure consistently.
