### 3. Reading Guide: Google Cloud Database Selection Guide
**High-Yield Concepts:**
1. **Cloud SQL:** Best for local/regional monolithic applications (MySQL, Postgres, SQL Server).
2. **Cloud Spanner:** Best for global, horizontally scalable relational databases with >99.999% availability.
3. **Cloud Bigtable:** Best for massive, single-key, high-throughput analytics and IoT data (NoSQL wide-column).
4. **Firestore:** Best for mobile/web apps requiring offline sync and document-based NoSQL storage.
