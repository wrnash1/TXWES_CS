## 3. Updated Online Syllabus
**Course:** CIS 4327 - Database Administration
**Instructor:** Professor Nash
**Format:** 100% Online Asynchronous
**Target Certification:** Google Cloud Professional Cloud Database Engineer

**Course Description:**
An advanced study of database administration in a cloud-native environment. Topics include selecting the appropriate database service (Relational vs. NoSQL), configuring High Availability (HA) and Disaster Recovery (DR), continuous data migration (DMS), and securing databases using IAM.

**Required Materials:**
1. **Zero Textbook Cost (ZTC):** All required reading materials, official Google Cloud documentation, and study guides are provided completely free within the Blackboard course modules or via the `ZTC_OER_Reading_Materials.md` document. No textbook purchase is required.
2. Active Google Cloud Student Account (Credits provided by the university).

**Course Objectives:**
By the end of this course, students will be able to:
1. Design scalable and highly available cloud database solutions.
2. Manage and operate database solutions, including backups and Point-in-Time Recovery.
3. Migrate data solutions from on-premises to Google Cloud seamlessly.
4. Implement database security, IAM least privilege, and compliance monitoring.
