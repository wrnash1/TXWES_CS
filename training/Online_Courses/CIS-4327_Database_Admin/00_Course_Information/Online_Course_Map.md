# Online Course Map: CIS 4327 - Database Administration
**Target Certification:** Google Cloud Professional Cloud Database Engineer

## Overview
This online course map aligns CIS 4327 with the Google Cloud Database Engineer certification. The asynchronous design utilizes Google Cloud Skills Boost / GCP student credits for cloud labs, while core concepts are delivered in micro-learning bursts supported by Distractor Analysis in all assessments.

---

## Module Architecture

### Module 1: Designing Scalable Cloud Database Solutions (Domain 1)
- **Learning Objectives:** Evaluate and choose the appropriate Google Cloud database (Cloud SQL, Spanner, Bigtable, Firestore).
- **Micro-Learning Video Assets (5-7 mins):**
  - *Video 1.1: Relational vs. Non-Relational on Google Cloud*
  - *Video 1.2: Deep Dive: Cloud SQL vs Cloud Spanner* (Includes Alt-text for horizontal vs vertical scaling diagrams)
- **Reading Guide:** Google Cloud Documentation - Database Selection Guide.
- **Lab Assignment:** Provision a Cloud SQL instance and connect via Cloud Shell.
- **Assessment:** Quiz 1 (Domain 1) with Distractor Analysis.

### Module 2: Managing and Operating Database Solutions (Domain 2)
- **Learning Objectives:** Configure high availability, backups, and disaster recovery.
- **Micro-Learning Video Assets:**
  - *Video 2.1: Configuring High Availability (HA) in Cloud SQL*
  - *Video 2.2: Backup and Recovery Strategies* (Includes Alt-text for failover architecture diagrams)
- **Reading Guide:** Google Cloud Documentation - Cloud SQL HA and Backups.
- **Lab Assignment:** Perform a point-in-time recovery on a Cloud SQL instance.
- **Assessment:** Quiz 2 (Domain 2) with Distractor Analysis.

### Module 3: Migrating Data Solutions (Domain 3)
- **Learning Objectives:** Migrate on-premises databases to Google Cloud using Database Migration Service (DMS).
- **Micro-Learning Video Assets:**
  - *Video 3.1: Introduction to Google Cloud DMS*
  - *Video 3.2: Homogeneous vs Heterogeneous Migrations*
- **Reading Guide:** Google Cloud Documentation - Database Migration Service.
- **Lab Assignment:** Execute a continuous migration job from a local MySQL VM to Cloud SQL.
- **Assessment:** Quiz 3 (Domain 3) with Distractor Analysis.

### Module 4: Security, Compliance, and Monitoring (Domain 4)
- **Learning Objectives:** Implement IAM roles, encryption (CMEK), and monitor with Cloud Monitoring.
- **Micro-Learning Video Assets:**
  - *Video 4.1: IAM Roles for Database Administrators* (Includes Alt-text for IAM policy structure)
  - *Video 4.2: Query Insights and Cloud Monitoring*
- **Reading Guide:** Google Cloud Documentation - Security and Monitoring.
- **Lab Assignment:** Configure IAM access and set up alerting policies for high CPU usage.
- **Assessment:** Quiz 4 (Domain 4) with Distractor Analysis.

---

## Pedagogical Standards Check
- [x] **Micro-Learning:** All videos capped at 5-7 minutes.
- [x] **Question Banks:** All quizzes map to Google Cloud domains and use Distractor Analysis.
- [x] **Accessibility:** Visual aids in videos require explicitly written Alt-text scripts.
- [x] **Labs:** Tuned for Google Cloud Student Credits / Qwiklabs.
